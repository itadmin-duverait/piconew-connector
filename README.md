# PicoNew Connect

Router installer package for connecting PicoBrew machines to PicoNew.

## Download

Download the latest installer from the [Releases](https://github.com/itadmin-duverait/piconew-connector/releases) page.

## Supported hardware

- GL.iNet GL-SFT1200 (validated)
- Other GL.iNet / OpenWRT routers may work

## Supported devices

- PicoBrew Z
- Pico C / Pico S / Pico Pro
- PicoFerm

## Setup instructions

See [piconew.com/connect](https://www.piconew.com/connect) for full setup instructions.
